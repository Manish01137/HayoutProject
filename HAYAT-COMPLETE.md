# Hayat — Complete Project Document

A single document containing the entire project: all code, all setup commands, all configuration files. Copy-paste the commands below to run it on your machine.

> **Not a medical device.** This is a research prototype and assistive communication aid. It does not diagnose, predict, or replace clinical care.

---

## What's in the system

1. **Web app** (React PWA) — child's icon-based communication interface + parent dashboard
2. **Backend API** (FastAPI) — receives events from bracelet and web app, pushes notifications
3. **Bracelet firmware** (ESP32 Arduino) — reads biometric sensors, computes stress, posts events

All three pieces work together, but the **web app runs standalone** even without the hardware — it has a built-in bracelet simulator so you can demonstrate the full pipeline.

---

## Prerequisites

Install these once on your machine:

- **Python 3.9+** — [python.org/downloads](https://www.python.org/downloads/)
- **Node.js 18+** — [nodejs.org](https://nodejs.org/)
- **Arduino IDE 2.x** (only if flashing the real ESP32) — [arduino.cc/en/software](https://www.arduino.cc/en/software)

Check that they're installed:


## Quick start — run everything in 5 minutes

Open three terminal windows. Paste one block into each.

### Terminal 1 — Backend (FastAPI)

```bash
# Create the project folder
mkdir -p hayat-project/backend
cd hayat-project/backend

# Create Python virtual environment
python3 -m venv venv

# Activate it
source venv/bin/activate          # macOS / Linux
# venv\Scripts\activate            # Windows (use this instead on Windows)

# Install dependencies
pip install fastapi "uvicorn[standard]" sqlalchemy pydantic python-multipart

# Save the backend code (see "File: backend.py" section below) as backend.py in this folder
# Then run:
uvicorn backend:app --host 0.0.0.0 --port 8000 --reload
```

You should see `Uvicorn running on http://0.0.0.0:8000`. Open `http://localhost:8000/docs` in your browser — you'll see the interactive API documentation.

### Terminal 2 — Web app (React + Vite)

```bash
# Go back to the project root
cd hayat-project

# Scaffold a React + Vite project
npm create vite@latest web -- --template react
cd web

# Install dependencies
npm install
npm install lucide-react recharts
npm install -D tailwindcss@3 postcss autoprefixer

# Initialize Tailwind
npx tailwindcss init -p

# Now replace three files with the contents shown in the "Frontend configuration" section below:
#   - tailwind.config.js
#   - src/index.css
#   - src/App.jsx      (paste the whole hayat-app.jsx contents here)

# Start the dev server
npm run dev
```

Open `http://localhost:5173` in your browser. The app works immediately — no need to wait for the backend or hardware.

### Terminal 3 — Bracelet firmware (optional)

You don't need a terminal for this — open the **Arduino IDE**, paste `firmware.ino` into a new sketch, edit the WiFi and server IP at the top, and click **Upload**. Detailed steps are in the "ESP32 firmware" section.

```python
"""
Hayat Backend API
-----------------
Receives stress events from the ESP32 bracelet and icon taps from the web app.
Pushes notifications to the parent's phone.

Run:
  uvicorn backend:app --host 0.0.0.0 --port 8000 --reload
"""

from datetime import datetime
from typing import List, Optional, Literal
import asyncio

from fastapi import FastAPI, HTTPException, WebSocket, WebSocketDisconnect, Depends
from fastapi.middleware.cors import CORSMiddleware
from pydantic import BaseModel, Field
from sqlalchemy import create_engine, Column, Integer, String, Float, Boolean, DateTime
from sqlalchemy.orm import declarative_base, sessionmaker, Session

# ------------------------ Database ------------------------

DATABASE_URL = "sqlite:///./hayat.db"
engine = create_engine(DATABASE_URL, connect_args={"check_same_thread": False})
SessionLocal = sessionmaker(bind=engine)
Base = declarative_base()


class Event(Base):
    __tablename__ = "events"
    id = Column(Integer, primary_key=True, index=True)
    child_id = Column(String, index=True, default="default")
    type = Column(String, index=True)
    category = Column(String, nullable=True)
    item_id = Column(String, nullable=True)
    label = Column(String, nullable=True)
    label_ar = Column(String, nullable=True)
    level = Column(Float, nullable=True)
    heart_rate = Column(Float, nullable=True)
    hrv = Column(Float, nullable=True)
    acknowledged = Column(Boolean, default=False)
    timestamp = Column(DateTime, default=datetime.utcnow, index=True)


class ChildProfile(Base):
    __tablename__ = "children"
    id = Column(String, primary_key=True)
    name = Column(String)
    parent_device_token = Column(String, nullable=True)
    parent_phone = Column(String, nullable=True)
    stress_threshold = Column(Integer, default=75)
    baseline_hr = Column(Float, nullable=True)
    baseline_hrv = Column(Float, nullable=True)


Base.metadata.create_all(bind=engine)


def get_db():
    db = SessionLocal()
    try:
        yield db
    finally:
        db.close()


# ------------------------ Schemas ------------------------

class TapIn(BaseModel):
    child_id: str = "default"
    category: Literal["feelings", "needs", "requests", "sensory"]
    item_id: str
    label: str
    label_ar: Optional[str] = None


class StressIn(BaseModel):
    child_id: str = "default"
    level: float = Field(..., ge=0, le=100)
    heart_rate: float
    hrv: Optional[float] = None
    movement_intensity: Optional[float] = None


class EventOut(BaseModel):
    id: int
    child_id: str
    type: str
    category: Optional[str]
    item_id: Optional[str]
    label: Optional[str]
    label_ar: Optional[str]
    level: Optional[float]
    heart_rate: Optional[float]
    hrv: Optional[float]
    acknowledged: bool
    timestamp: datetime

    class Config:
        from_attributes = True


# ------------------------ WebSocket manager ------------------------

class ConnectionManager:
    def __init__(self):
        self.active: List[WebSocket] = []

    async def connect(self, ws: WebSocket):
        await ws.accept()
        self.active.append(ws)

    def disconnect(self, ws: WebSocket):
        if ws in self.active:
            self.active.remove(ws)

    async def broadcast(self, payload: dict):
        dead = []
        for ws in self.active:
            try:
                await ws.send_json(payload)
            except Exception:
                dead.append(ws)
        for ws in dead:
            self.disconnect(ws)


manager = ConnectionManager()

# ------------------------ App ------------------------

app = FastAPI(title="Hayat API", version="0.1.0")

app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],
    allow_methods=["*"],
    allow_headers=["*"],
)


@app.get("/")
def root():
    return {"service": "Hayat API", "status": "ok"}


@app.post("/api/taps", response_model=EventOut)
async def create_tap(tap: TapIn, db: Session = Depends(get_db)):
    event = Event(
        child_id=tap.child_id,
        type="tap",
        category=tap.category,
        item_id=tap.item_id,
        label=tap.label,
        label_ar=tap.label_ar,
    )
    db.add(event)
    db.commit()
    db.refresh(event)

    title = "Your child has a message"
    body = f"{tap.label}"
    asyncio.create_task(send_push_notification(tap.child_id, title, body, db))
    await manager.broadcast({
        "type": "new_event",
        "event": EventOut.model_validate(event).model_dump(mode="json")
    })
    return event


@app.post("/api/stress", response_model=EventOut)
async def create_stress_event(data: StressIn, db: Session = Depends(get_db)):
    child = db.query(ChildProfile).filter(ChildProfile.id == data.child_id).first()
    threshold = child.stress_threshold if child else 75
    push = data.level >= threshold

    event = Event(
        child_id=data.child_id,
        type="stress",
        level=data.level,
        heart_rate=data.heart_rate,
        hrv=data.hrv,
    )
    db.add(event)
    db.commit()
    db.refresh(event)

    if push:
        title = "Stress level elevated"
        body = f"Stress {int(data.level)}% · HR {int(data.heart_rate)} bpm"
        asyncio.create_task(send_push_notification(data.child_id, title, body, db))

    await manager.broadcast({
        "type": "new_event",
        "event": EventOut.model_validate(event).model_dump(mode="json")
    })
    return event


@app.get("/api/events", response_model=List[EventOut])
def list_events(
    child_id: str = "default",
    event_type: Optional[str] = None,
    limit: int = 100,
    db: Session = Depends(get_db),
):
    q = db.query(Event).filter(Event.child_id == child_id)
    if event_type:
        q = q.filter(Event.type == event_type)
    return q.order_by(Event.timestamp.desc()).limit(limit).all()


@app.post("/api/events/{event_id}/ack")
def acknowledge(event_id: int, db: Session = Depends(get_db)):
    event = db.query(Event).filter(Event.id == event_id).first()
    if not event:
        raise HTTPException(status_code=404, detail="not found")
    event.acknowledged = True
    db.commit()
    return {"ok": True}


@app.get("/api/children/{child_id}")
def get_child(child_id: str, db: Session = Depends(get_db)):
    child = db.query(ChildProfile).filter(ChildProfile.id == child_id).first()
    if not child:
        raise HTTPException(status_code=404, detail="not found")
    return {
        "id": child.id,
        "name": child.name,
        "stress_threshold": child.stress_threshold,
        "baseline_hr": child.baseline_hr,
        "baseline_hrv": child.baseline_hrv,
    }


@app.post("/api/children")
def upsert_child(
    child_id: str,
    name: str,
    stress_threshold: int = 75,
    parent_device_token: Optional[str] = None,
    parent_phone: Optional[str] = None,
    db: Session = Depends(get_db),
):
    child = db.query(ChildProfile).filter(ChildProfile.id == child_id).first()
    if child:
        child.name = name
        child.stress_threshold = stress_threshold
        if parent_device_token:
            child.parent_device_token = parent_device_token
        if parent_phone:
            child.parent_phone = parent_phone
    else:
        child = ChildProfile(
            id=child_id,
            name=name,
            stress_threshold=stress_threshold,
            parent_device_token=parent_device_token,
            parent_phone=parent_phone,
        )
        db.add(child)
    db.commit()
    return {"ok": True, "child_id": child_id}


@app.websocket("/ws")
async def websocket_endpoint(ws: WebSocket):
    await manager.connect(ws)
    try:
        while True:
            await ws.receive_text()
    except WebSocketDisconnect:
        manager.disconnect(ws)
```

## Hardware wiring

| Component | Pin on ESP32 | Notes |
|---|---|---|
| MAX30102 VIN | 3.3V | Pulse oximeter |
| MAX30102 GND | GND | |
| MAX30102 SDA | GPIO 21 | I²C data |
| MAX30102 SCL | GPIO 22 | I²C clock |
| MPU6050 VIN | 3.3V | Same I²C bus as MAX30102 |
| MPU6050 GND | GND | |
| MPU6050 SDA | GPIO 21 | Shared |
| MPU6050 SCL | GPIO 22 | Shared |
| Vibration motor + | GPIO 25 | Optional haptic |
| Vibration motor − | GND | |
| LiPo battery + | ESP32 VIN (via TP4056) | |
| LiPo battery − | GND | |

## Arduino IDE libraries

Open **Tools → Manage Libraries** and install:

- `SparkFun MAX3010x Pulse and Proximity Sensor Library`
- `Adafruit MPU6050`
- `Adafruit Unified Sensor` (dependency)
- `Adafruit BusIO` (dependency)
- `ArduinoJson` by Benoit Blanchon

Install the **ESP32 board support**: File → Preferences → add `https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json` to Additional Boards, then Tools → Board Manager → install "esp32".

## firmware.ino source

```cpp
#include <Wire.h>
#include <WiFi.h>
#include <HTTPClient.h>
#include <ArduinoJson.h>
#include "MAX30105.h"
#include "heartRate.h"
#include <Adafruit_MPU6050.h>
#include <Adafruit_Sensor.h>

// ========= CONFIG — EDIT THESE =========
const char* WIFI_SSID     = "YOUR_WIFI_SSID";
const char* WIFI_PASSWORD = "YOUR_WIFI_PASSWORD";
const char* API_URL       = "http://YOUR_SERVER_IP:8000/api/stress";
const char* CHILD_ID      = "default";

const int STRESS_THRESHOLD   = 75;
const int SUSTAIN_SECONDS    = 15;
const int CALIBRATION_SEC    = 120;
const int VIBRATION_PIN      = 25;
const int POST_COOLDOWN_SEC  = 60;

// ========= SENSORS =========
MAX30105 pulseOx;
Adafruit_MPU6050 mpu;

const int RATE_SIZE = 8;
byte rates[RATE_SIZE];
byte rateSpot = 0;
long lastBeat = 0;

float rrIntervals[20];
int   rrCount = 0;

float baselineHR  = 0;
float baselineHRV = 0;
bool  calibrated  = false;
unsigned long calibStart = 0;

float currentHR  = 0;
float currentHRV = 0;
float currentMotion = 0;

int  aboveThresholdSeconds = 0;
unsigned long lastAlertMs  = 0;

void connectWifi() {
  Serial.print("Wi-Fi connecting");
  WiFi.begin(WIFI_SSID, WIFI_PASSWORD);
  int tries = 0;
  while (WiFi.status() != WL_CONNECTED && tries < 40) {
    delay(500);
    Serial.print(".");
    tries++;
  }
  Serial.println();
  if (WiFi.status() == WL_CONNECTED) {
    Serial.print("Connected. IP: ");
    Serial.println(WiFi.localIP());
  }
}

void initSensors() {
  Wire.begin();
  if (!pulseOx.begin(Wire, I2C_SPEED_FAST)) {
    Serial.println("MAX30102 not found.");
    while (1) delay(100);
  }
  pulseOx.setup();
  pulseOx.setPulseAmplitudeRed(0x0A);
  pulseOx.setPulseAmplitudeGreen(0);

  if (!mpu.begin()) {
    Serial.println("MPU6050 not found.");
    while (1) delay(100);
  }
  mpu.setAccelerometerRange(MPU6050_RANGE_4_G);
  mpu.setGyroRange(MPU6050_RANGE_500_DEG);
  mpu.setFilterBandwidth(MPU6050_BAND_21_HZ);
}

void readPulse() {
  long irValue = pulseOx.getIR();
  if (checkForBeat(irValue)) {
    long delta = millis() - lastBeat;
    lastBeat = millis();
    float bpm = 60.0 / (delta / 1000.0);
    if (bpm > 40 && bpm < 180) {
      rates[rateSpot++] = (byte)bpm;
      rateSpot %= RATE_SIZE;
      float avg = 0;
      for (byte i = 0; i < RATE_SIZE; i++) avg += rates[i];
      currentHR = avg / RATE_SIZE;
      if (rrCount < 20) rrIntervals[rrCount++] = delta;
      else {
        for (int i = 0; i < 19; i++) rrIntervals[i] = rrIntervals[i+1];
        rrIntervals[19] = delta;
      }
      if (rrCount >= 5) {
        float sumSq = 0;
        int n = rrCount - 1;
        for (int i = 0; i < n; i++) {
          float d = rrIntervals[i+1] - rrIntervals[i];
          sumSq += d * d;
        }
        currentHRV = sqrt(sumSq / n);
      }
    }
  }
}

void readMotion() {
  sensors_event_t a, g, temp;
  mpu.getEvent(&a, &g, &temp);
  float m = sqrt(a.acceleration.x*a.acceleration.x +
                 a.acceleration.y*a.acceleration.y +
                 a.acceleration.z*a.acceleration.z);
  currentMotion = 0.7 * currentMotion + 0.3 * abs(m - 9.8);
}

float computeStressScore() {
  if (!calibrated) return 0;
  float hrDev = (currentHR - baselineHR) / max(baselineHR * 0.3f, 10.0f);
  hrDev = constrain(hrDev, 0.0f, 1.0f);
  float hrvDrop = 0;
  if (baselineHRV > 0 && currentHRV > 0) {
    hrvDrop = (baselineHRV - currentHRV) / baselineHRV;
    hrvDrop = constrain(hrvDrop, 0.0f, 1.0f);
  }
  float motionF = constrain(currentMotion / 5.0f, 0.0f, 1.0f);
  float score = (0.45 * hrDev + 0.35 * hrvDrop + 0.20 * motionF) * 100.0;
  return constrain(score, 0.0f, 100.0f);
}

void maybeCalibrate() {
  if (calibrated) return;
  unsigned long elapsed = (millis() - calibStart) / 1000;
  static float hrSum = 0, hrvSum = 0;
  static int hrN = 0, hrvN = 0;
  if (currentHR > 0)  { hrSum  += currentHR;  hrN++;  }
  if (currentHRV > 0) { hrvSum += currentHRV; hrvN++; }
  if (elapsed >= CALIBRATION_SEC && hrN > 30) {
    baselineHR  = hrSum / hrN;
    baselineHRV = hrvN > 0 ? hrvSum / hrvN : 40.0;
    calibrated  = true;
    Serial.printf("Calibrated. HR=%.1f HRV=%.1f\n", baselineHR, baselineHRV);
  }
}

void postStressEvent(float score) {
  if (WiFi.status() != WL_CONNECTED) connectWifi();
  if (WiFi.status() != WL_CONNECTED) return;

  HTTPClient http;
  http.begin(API_URL);
  http.addHeader("Content-Type", "application/json");

  StaticJsonDocument<256> doc;
  doc["child_id"] = CHILD_ID;
  doc["level"] = score;
  doc["heart_rate"] = currentHR;
  doc["hrv"] = currentHRV;
  doc["movement_intensity"] = currentMotion;

  String body;
  serializeJson(doc, body);
  int code = http.POST(body);
  Serial.printf("POST -> %d\n", code);
  http.end();

  pinMode(VIBRATION_PIN, OUTPUT);
  digitalWrite(VIBRATION_PIN, HIGH);
  delay(250);
  digitalWrite(VIBRATION_PIN, LOW);
}

void setup() {
  Serial.begin(115200);
  delay(500);
  pinMode(VIBRATION_PIN, OUTPUT);
  digitalWrite(VIBRATION_PIN, LOW);
  initSensors();
  connectWifi();
  calibStart = millis();
}

void loop() {
  readPulse();
  static unsigned long lastTick = 0;
  if (millis() - lastTick < 1000) return;
  lastTick = millis();

  readMotion();
  maybeCalibrate();

  float score = computeStressScore();
  Serial.printf("HR=%.0f HRV=%.0f motion=%.2f score=%.0f\n",
                currentHR, currentHRV, currentMotion, score);

  if (calibrated) {
    if (score >= STRESS_THRESHOLD) aboveThresholdSeconds++;
    else aboveThresholdSeconds = 0;

    bool cooldownOK = (millis() - lastAlertMs) / 1000 > POST_COOLDOWN_SEC;
    if (aboveThresholdSeconds >= SUSTAIN_SECONDS && cooldownOK) {
      postStressEvent(score);
      lastAlertMs = millis();
      aboveThresholdSeconds = 0;
    }
  }
}
```

The React code is too long to inline cleanly here (~700 lines). It's provided as a separate file — paste it whole into `web/src/App.jsx` replacing the default Vite App.jsx.

Key points when integrating:
- It uses `import` statements for `react`, `lucide-react`, and `recharts` — all three are installed via the npm commands in Quick Start.
- It uses `window.storage` for persistence. In a standard Vite project, `window.storage` is not defined — replace the four `window.storage.*` helper calls at the top (`loadEvents`, `saveEvents`, `loadSettings`, `saveSettings`) with `localStorage`. Two small edits, shown below.

## Replacing `window.storage` with `localStorage`

Near the top of `App.jsx`, find the `STORAGE` section and replace it with:

```js
const loadEvents = async () => {
  try {
    const v = localStorage.getItem('events');
    return v ? JSON.parse(v) : [];
  } catch { return []; }
};

const saveEvents = async (events) => {
  try {
    localStorage.setItem('events', JSON.stringify(events.slice(0, 500)));
  } catch {}
};

const loadSettings = async () => {
  try {
    const v = localStorage.getItem('settings');
    return v ? { ...DEFAULT_SETTINGS, ...JSON.parse(v) } : DEFAULT_SETTINGS;
  } catch { return DEFAULT_SETTINGS; }
};

const saveSettings = async (s) => {
  try {
    localStorage.setItem('settings', JSON.stringify(s));
  } catch {}
};
```

That's the only code edit needed. Everything else works as-is.

# Common problems

**`uvicorn: command not found`** — You forgot to activate the virtualenv. Run `source venv/bin/activate` (macOS/Linux) or `venv\Scripts\activate` (Windows).

**Port 8000 already in use** — Something else is using it. Run `uvicorn backend:app --port 8001 --reload` and update the API URL in the frontend and firmware to match.

**`npm ERR! Cannot find module`** — Run `npm install` inside the `web/` folder.

**Tailwind classes not applying** — Double-check `tailwind.config.js` has the `content` paths pointing to `./src/**/*.{js,ts,jsx,tsx}`, and that `index.css` imports the three `@tailwind` directives, and that `main.jsx` imports `./index.css`.

**ESP32 MAX30102 not found** — Check the I²C wiring. Run an I²C scanner sketch first to confirm both devices show up (MAX30102 at `0x57`, MPU6050 at `0x68`).

**CORS error from browser** — The backend already has `allow_origins=["*"]`, so this shouldn't happen. If it does, check the API URL includes the correct protocol (`http://` not `https://`) and port.

# Clinical and ethical considerations

A project working with vulnerable children is different from other software projects.

- **This is not diagnostic.** The honest framing is that the system flags elevated physiological arousal that correlates with distress, and gives the child a channel to communicate. It does not predict meltdowns.
- **Validate with professionals.** Pilot alongside an occupational therapist or developmental pediatrician.
- **Biometric child data is highly sensitive.** Obtain informed parental consent, scope collection to the minimum necessary, encrypt in transit and at rest.
- **Respect child agency.** The bracelet must be removable by the child; the app must have an obvious exit.
- **Cultural adaptation.** Test icons, colors, and wording with the actual families you're targeting.
